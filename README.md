# backend-project
A Simple CRUD Project For Me:^

## Clone this Repo
```
git clone https://github.com/FIlham/backend-project
```

## Installation
```js
npm install
```

## Usage
Development Mode
```js
npm run dev
```
Production Mode
```js
npm start
```

## Documentation ##
<hr>
• Main Route
<br>
/

• Add User
<br>
/user/addUser/ [POST]
<br>
Data post must be JSON
```json
{
  "name": "filham",
  "email": "filham@gmail.com"
}
```

• Get User
<br>
/user/getUser/ex@gmail.com [GET]
<br>

• Update/edit User
<br>
/user/editUser/ex@gmail.com [PATCH]
<br>
Data edit must be JSON
```json
{
  "name": "ilham",
  "email": "ilham@gmail.com"
}
```

• Delete User
<br>
/user/deleteUser/ex@gmail.com [DELETE]
